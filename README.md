# 🖼️ Image Classification with CNN

## 📌 Project Overview
Classifying fashion items using a Convolutional 
Neural Network (CNN) trained on the 
Fashion MNIST dataset.

## 📊 Dataset
- Fashion MNIST Dataset
- 10 categories of clothing items
- 70,000 images (60,000 train / 10,000 test)

## 🛠️ Tools & Technologies
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## ⚙️ Steps
1. Loading & Exploring Fashion MNIST
2. Data Preprocessing & Normalization
3. Building CNN Architecture
4. Training the Model
5. Model Evaluation & Visualization

## 📈 Model Evaluation
- Accuracy
- Confusion Matrix
- Loss & Accuracy Curves

## 💼 Business Impact
Automates fashion item recognition,
useful for e-commerce and retail industries.
## 📈 Model Results

| Metric | Value |
|--------|-------|
| Test Accuracy | 91% |
| Dataset | Fashion MNIST |
| Architecture | CNN |
| Epochs | 10 |
